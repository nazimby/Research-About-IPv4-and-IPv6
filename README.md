# Research-About-IPv4-and-IPv6
Araşdırmamda toxunduğum əsas texniki nüanslar:

    IPv4 Təkamülü və CIDR: Köhnəlmiş sinifli (Classful) şəbəkə yanaşmasından IP israfının qarşısını alan CIDR (Classless Inter-Domain Routing) sisteminə keçid.

    İnternetin Xilaskarı - NAT: Məhdud sayda olan Public IP-lərin arxasında daxili şəbəkədəki minlərlə Private IP-ni gizlədərək qlobal şəbəkəyə çıxışı təmin edən NAT mexanizmi.

    IPv6 İnqilabı: IPv6-da Broadcast anlayışının Multicast ilə əvəz olunması və ARP protokolunun fəaliyyətini tamamilə NDP (Neighbor Discovery Protocol) ilə əvəzləməsi.

    Avtomatik Konfiqurasiya: IPv6 mühitində cihazların IP alma mexanizmləri, xüsusilə SLAAC və DHCPv6 (Stateful/Stateless) arasındakı fərqlər.
